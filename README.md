## On-call Handover Tool

A lightweight internal tool for capturing and reviewing on-call handover notes. The goal is to make shift transitions smoother by clearly documenting what happened and what needs attention next.

This project was built as a take-home assignment, focusing on quickly turning a vague request into a working full-stack prototype using AI-assisted development.

### Tech stack

- HTML, CSS, JavaScript  
- Python (FastAPI)  
- JSON-based persistence
