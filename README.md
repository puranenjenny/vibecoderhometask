# On-call Handover Tool

A lightweight internal tool for capturing and reviewing on-call handover notes. The goal is to make shift transitions smoother by clearly documenting what happened and what needs attention next.

This project was built as a take-home assignment, focusing on quickly turning a vague request into a working full-stack prototype.

## Tech stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python (FastAPI)
- Storage: JSON file

### How to run

1. Install dependencies:
   pip install fastapi uvicorn

2. Start backend:
   cd backend
   uvicorn main:app --reload

3. Open the frontend in a browser
